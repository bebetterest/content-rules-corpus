                                                01视频绿屏/花屏、卡顿/资源占用高如何解决？

                                                绿屏/花屏问题排查：

                                                1. 检查显卡驱动是否过旧，更新显卡驱动；

                                                2. 检查是否浏览器问题，更换浏览器尝试；

                                                3. 检查是否非系统解码器问题，或在播放器设置调整播放策略；

                                                4. 如果以上方法还是解决不了问题，请联系客服。


                                                卡顿/资源占用高排查：

                                                1. 查看 Chrome / Edge / Chromium 内核浏览器「图形加速功能」是否开启

                                                可通过 Chrome / Edge / Chromium 内核浏览器中「设置」-「系统」，或者输入框直接输入chrome://settings/system 进行设置

                                                 a. 开启「图形加速功能」（ 推荐 ），开启后需重启浏览器

                                                 b. 重启「图形加速功能」：设置功能关闭，重启浏览器，再设置功能开启，再次重启浏览器即可生效

                                                2. 查看 Chrome / Edge 等浏览器默认「性能设置」，可通过 Chrome 设置 - 性能，或者输入框直接输入chrome://settings/performance 进行设置

                                                3. 检查是否网络问题。检测工具：

                                                 a. 网页端https://www.bilibili.com/blackboard/video-diagnostics.html

                                                 b. APPhttps://www.bilibili.com/blackboard/app-speed-test.html

                                                4. 按需调整 Web 播放器内设置来降低 CPU 负载，例如调节弹幕显示区域、关闭智能防挡弹幕、更多设置调整播放策略等。

                                                5. 查看 Chrome / Edge 等浏览器使用中影响性能页面（ 浏览器 Tab ），可通过 Chrome 更多工具 - 任务管理器查看

                                                6. 通过 Windows 系统「任务管理器」或 macOS「活动监控器」来查看整体系统性能占用，可以依次尝试以下对比验证 Web 页面 CPU 性能问题根源：

                                                 尝试逐个关闭浏览器页面 Tab 对比

                                                 尝试关闭浏览器插件对比

                                                 尝试使用当前浏览器的隐身 / 无痕模式对比

                                                 尝试使用另一个浏览器对比，优先建议 Chrome / Edge 等

                                                如能复现特定页面性能占用异常情况，可以联系客服并描述复现场景及操作，反馈时请告知您的浏览器型号和相应问题的截图或录屏。（ Web 播放场景可以点击右上角反馈提交日志 ）


                                                02视频无法加载怎么办？

                                                1. 连接到视频源获取信息失败，可能是您的网络问题或bilibili服务器故障。检测工具：

                                                http://www.bilibili.com/blackboard/diagnostics.html

                                                2. 如果以上方法还是解决不了问题，请联系客服。


                                                03如果邮箱没有收到激活邮件怎么办？

                                                邮件发送具有一定的延迟请等待一段时间后再次查询；检查邮件垃圾箱中是否有激活邮件；可以在会员中心点击再次发送激活邮件；尝试更换邮箱再进行激活；
